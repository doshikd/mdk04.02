Создание установочного образа системы:
sudo add-apt-repository ppa:cubic-wizard/release - добавляет сторонний репозиторий, где хранится актуальная версия программы Cubic
sudo apt update
sudo apt install cubic -y

заходим в приложение -> выбираем директорию -> выбираем iso файл -> загружаем виртуальную среду -> (добавляем свои программы через sudo apt install) -> собираем ISO
